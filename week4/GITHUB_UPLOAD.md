# Upload to GitHub - Next Steps

After committing your code, run these commands:

## 1. Add GitHub Remote
Replace YOUR_USERNAME with your actual GitHub username:
```bash
git remote add origin https://github.com/YOUR_USERNAME/redux-shopping-cart.git
```

## 2. Push to GitHub
```bash
git branch -M main
git push -u origin main
```
